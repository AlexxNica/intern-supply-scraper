### Installation

  - Clone repository
  
    ```
    $ git clone https://github.com/kshvmdn/intern-supply-scraper.git
    $ cd intern-supply-scraper
    ```

  - Install dependencies

    ```
    $ pip install -r ./requirements.txt
    ```
